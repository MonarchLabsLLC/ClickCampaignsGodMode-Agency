# ClickCampaigns God Mode — Agency

Your AI-powered agency workspace. 22 marketing specialists, 73 skill files, 26 funnel types. Each client gets their own folder.

**New here? Open `START-HERE.md` for step-by-step instructions.**

## What's in This Folder

| File | Who It's For | What It Does |
|------|-------------|-------------|
| `START-HERE.md` | You | Step-by-step setup instructions |
| `CLAUDE.md` | Claude Code / Cursor | Tells the AI how God Mode works (auto-loaded) |
| `AGENTS.md` | Codex / other AI tools | Same instructions for non-Claude tools (auto-loaded) |
| `.env.example` | You | Template for optional API keys |
| `.gitignore` | Git | Keeps your `.env` keys out of GitHub |

## Folder Structure

```
clients/                  One folder per client
  [client-name]/
    brand-kit/            Client's brand files
      knowledge-base/     Product info, audience research
      style-guide/        Colors, fonts, logos
    campaigns/            Client's campaign workspaces
    output-assets/        Everything Alex creates for this client
      html/               Funnel pages
      emails/             Email sequences
      ads/                Ad copy
      documents/          Scripts, letters, guides
      presentations/      Webinar content
      pdfs/               Lead magnets
      images/             Photos and graphics
```
